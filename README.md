# Help Desk Ticketing System

## Project Overview
The Help Desk Ticketing System is a Java console application developed to help a small IT support team manage support requests. Users can create tickets, view existing tickets, update ticket statuses, and filter tickets based on their status.

## Features
* Create support tickets
* View all tickets
* Update ticket status
* Filter tickets by status (Open, In Progress, Closed)

## Technologies Used
* Java
* VS Code
* Git & GitHub

## Active Features

### add-ticket-filter
This feature branch introduces ticket filtering by status. It allows users to display only tickets with a selected status, making it easier for technicians to manage support requests.
